# Babki Compute

**Babki Compute**, также известный как **Babki_ON**, — экспериментальный проект распределённых вычислений и собственной Proof-of-Compute-инфраструктуры.

Идея проекта проста:

- владельцы компьютеров, серверов и GPU могут предоставлять свободные вычислительные ресурсы;
- пользователи могут отправлять задачи для обработки;
- задачи распределяются между вычислительными нодами;
- результаты проверяются;
- участники получают вознаграждение за полезную вычислительную работу.

> Проект находится в стадии перезапуска и разработки Proof of Concept.  
> Это не готовый коммерческий сервис и не публичная инвестиционная платформа.

---

## Текущий статус проекта

В 2025 году были проведены первые технические эксперименты:

- создан тестовый токен BABK;
- проверены тестовые переводы между кошельками;
- создан прототип подключения MetaMask;
- создан Flask API для передачи задач;
- создан Python-обработчик задач;
- протестирована обработка простых вычислительных заданий;
- протестирован ручной механизм выплаты вознаграждений;
- проведены эксперименты с собственной блокчейн-сетью;
- запущены тестовые validator и peer-ноды.

После продолжительного перерыва было принято решение не продолжать старую сборку вслепую, а заново проверить архитектуру, код, безопасность и документацию.

## Кому может быть интересен проект

### Владельцам вычислительных ресурсов

Участвовать потенциально смогут владельцы:

- обычных домашних компьютеров;
- игровых компьютеров;
- рабочих станций;
- серверов;
- GPU-серверов;
- майнинговых ферм;
- временно неиспользуемого оборудования.

На этапе PoC подключение ресурсов производится только после рассмотрения заявки через Telegram-бота.

### Пользователям с вычислительными задачами

В будущем через Babki Compute планируется обрабатывать:

- математические расчёты;
- научные и учебные задачи;
- перебор вариантов;
- обработку данных;
- моделирование;
- симуляции;
- задачи, которые можно разделить на независимые части;
- экспериментальные CPU/GPU-нагрузки.

На текущем этапе принимаются только заявки и описания потенциальных задач.

### Тестерам

Тестеры смогут помогать проверять:

- Telegram-бота;
- интерфейс сайта;
- передачу задач;
- работу очереди;
- подключение вычислительных нод;
- получение и отправку результатов;
- систему проверки;
- расчёт тестового вознаграждения;
- понятность документации.
